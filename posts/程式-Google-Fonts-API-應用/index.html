[程式]Google Fonts API 應用

tofudraft
2018-08-09
Angular, Google Fonts, 分享, 程式, 筆記

純分享。

Foncs

當初是在學UI，一個機會下，想到做這個應用。

以下為網頁實際圖

用了Angular作為前端框架
使用AngularCLI建置專案
介接應用Google Fonts

其網頁版功能相當完整
手機APP中有一款同為使用Google Fonts的應用－－Fontasy

優劣分析 … 我忘了

我達到提供的功能有：

  • 關鍵字搜尋字型
  • 字型類別篩選
  • 字型排序
  • 直接編輯檢視範例文字
  • 自訂範例文字色彩
  • 自訂範例文字大小

在Heroku上架一個DEMO
可以試試看
不過因為沒有使用付費功能，若超過一定時間沒有連線，會進入休眠
這時新進的連線，反應會比較慢一點

程式則放在GitHub上: chengfulin/foncs
並使用W3.CSS實作響應式設計
