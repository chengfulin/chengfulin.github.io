[程式]LINE Bot 猜數字

tofudraft
2018-08-10
LINE Message API, 分享, 程式, 筆記

純分享。

這也是之前做的。

那時候，剛好在玩 LINE Message API

同事提到有款狼人殺的

我從最基本的開始試試

想到以前大學程式玩過的猜數字

就用來看看 :P

加LINE

以下為相關指令：

  • @rule: 顯示說明
  • @game: 開始遊戲，並帶參數 4~9，表示要猜的數字長度
  • @giveup: 放棄不猜了，需要用手機看。會顯示確認提示

後端也是架在 Heroku 上，因為僅使用免費權限。當一段時間沒有連線，程式會進入休眠，這時的新連線會比較慢反應。

另外，原始碼也放到GitHub上了: chengfulin/guessnum-linebot
